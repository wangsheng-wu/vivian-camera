# Dual Camera Mount V1

## Overview
This is the first version of the dual camera mount for Vivian Camera.

- Designed for Raspberry Pi Camera Module 3
- Target baseline: 65mm (approximate human IPD)
- Material: Stainless steel plate

## Notes
- Manufacturing tolerances may affect actual alignment
- Camera modules themselves may introduce offsets
- Future versions may include adjustable alignment
